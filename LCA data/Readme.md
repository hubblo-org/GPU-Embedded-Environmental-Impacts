# LCA data

This fil contains the raw and processed data used to conduct the Life Cycle Assessment (LCA) of 8 GPUs and 1 GPU rack.

## File details by product

Each folder contains the following files: 

📁 EIME-Design.xlsx : EIME template (importable into the software and readable in Excel)

📁 EIME-Multi-Tracking.xlsx : EIME template (only importable into the software)

📁 EIME6.eime6 : EIMEIME template + details of each indicator for each architecture level

📁 Results.xlsx : Complete results

📁 BoM.xlsx : Bill Of Materials

📁 Life-stages results.xlsx : Impact results at the highest level (life cycle phase)

📁 Components results.xlsx : Impact results at the component level

📁 Documentation.xlsx : Documentation sheet
